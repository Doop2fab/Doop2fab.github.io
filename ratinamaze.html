Backtracking Rat-in-a-Maze Solver

This is a program which extends the Swing framework that solves Rat-in-a-Maze problems. It implements simple Backtracking by recursively determining if a space is safe to move to or not and, if the space in question is "un-safe", the rat backtracks. I have included 3 preset mazes, along with the ability to draw your own mazes.

Goal: In order to more completely understand how backtracking works, I endeavored to solve the Rat-in-a-Maze problem by implementing a recursive backtracking algorithm.

Challenges: The visual aspect of this program is arguably the most important, but the effectiveness and utility of the language it is written in was also something I considered. Because this program utilizes recursion, I needed to design the code to handle cases where the "Rat" would loop infinitely. I also had to include a way that this program could be downloaded and ran locally.

Solutions: Because this program is so visually-oriented, I put plenty of thought into how a user would interact with this program and designed a simple, intuitive UI to go along with it. I chose to write this program in Java, because I feel the language brings equal parts utility, readability, and effectiveness when one consideres the goal of this project. In order to solve the case of an infinite loop, I had the "Rat" mark where it had been before in the maze and backtrack if it could not find a correct path.

Key Takeaways: I completed this program with a more complete understanding of how backtracking algorithms work, how different components and tools within Java work together, how .jar files can be utilized, and how GitHub can be used to store and display code in a concise and readable way. I also have a deeper understanding of my abilities as a programmer, as I learned many troubleshooting skills.
