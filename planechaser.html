Planechaser

Planechaser is a Digital Companion App for Magic: The Gathering that allows players to play the Planechase format electronically! Create your own custom planes and enable or disable ones you and your playgroup find undesirable. Planechaser includes every official Planechase Anthology plane card complete with art and information. There's even a built-in counter widget to use for planes that require one!

Goal: There are many different formats and ways to play Magic: The Gathering with your friends, and Planechase is one format that my playgroup really enjoys. Planechase, also known as Planar Magic, is an official game variant first populated by Planechase and basically acts as an augmentation to other formats though players can also play decks specifically geared towards this format. This format requires that players own multiple Planechase cards which have not been printed for a significant amount of time, so it is very hard to actually play a game of Planechase these days. Thus, I set out to create a solution to this problem, and that's how Planechaser was created. With Planechaser, you do not need to own the physical cards to play; you only need open the app and begin!

Challenges: In the past, I had never really used an API to collect and utilize so much data in an application before. It certainly seemed daunting to me at first, because I was unsure of how to parse JSON files correctly. In addition, I was worried I would not be able to utilize all of the data collected from the API to its maximum potential.

Solutions: Building off of things I previously learned whilst putting together Backup Code Manager, I was able to incorporate Scryfall's robust API (https://scryfall.com/docs/api) which gives programmers access to any and all magic cards ever printed in Magic: The Gathering's history, complete with card text and images. Learning how to use Scryfall's API was a bit of a challenge, though, because it is extremely versitile and slightly more complicated than anything I have used in the past. Regardless, I perservered and learned how exactly to best implement this API.

Key Takeaways: Again, creating something that I can actually roll out to production has been an extremely robust and deep learning experience for me. I learned how to better manage versions on Google Play Console and roll out app bundles in the correct way. I also gained more experience working with others on a project, as I was lucky enough for my friend Charlie to create some assets for Planechaser. (Contact him at cmalachinski@outlook.com)
