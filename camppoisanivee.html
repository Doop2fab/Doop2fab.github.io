Camp Poisanivee

"Camp Poisanivee" reads in two files (one file contains campers to be processed through the program, and the other contains commands that can be used within the program) and uses a Binary Search Tree to maintain the set of campers enrolled in Camp Posanivee. It is then possible to run commands on the data to order and/or display it. There are commands within the Camper file to automatically sort and display data as a demonstration.

Goal: In order to get a better understanding of Binary Search Trees, File I/O, Linked Lists, and Traversals (Pre-order, In-Order, Post-Order), I created "Camp Poisanivee", which reads in data, orders the data with a Binary Search Tree, and reorders the data with Traversals so that it can de displayed back to the user.

Challenges: Implementing all four things that I wanted to learn (Binary Search Trees, File I/O, Linked Lists, and Traversals) was certainly a pertinent challenge. Also, understanding how I could implement them in a meaningful, real-world way added to the difficulty.

Solutions: In order to learn these things, I decided on writing a program that has real world applications - Camp Poisanivee could be used to order a roster for a Summer Camp or any other similar event that depends on a roster of people. The File I/O in "Camp Poisanivee" utilizes two files; one of them contains all Campers that want to enroll, and the other contains valid commands to order/reorder data and display it back to the user. The Binary Search Trees and Linked Lists were used to store the information on Campers and reorder them using the specified Traversal. And, of course, Traversals are used to reorder Campers that have been enrolled.

Key Takeaways: I finished this program with a more total understanding of how Binary Search Trees, File I/O, Linked Lists, and Traversals work in Java. Writing this program also demonstrated to me correct ways to not only implement different aspects of programming in a meaningful way, but also to understand what goes into writing useful, real-world code.
