Backup Code Manager

Backup Code Manager makes storing and accessing Two-Factor Authentication backup codes, locker combinations, one-time use passwords, and PIN codes fast and simple! View custom codes and categories securely and easily!

Goal: I really enjoy using my phone, and I also really enjoy having my accounts and social media secured by Two-Factor Authentication. While 2FA is nice, storing those backup/recovery codes and accessing them when I need them can be a bit of a pain. So, I designed Backup Code Manager to manage those backup/recovery codes. It kinda almost doubles as a way to store PINs, locker combos, etc., which I think is neat.

Challenges: When I started this project, I had Java experience under my belt but not Android development. Learning Android Studio was one of the largest roadblocks for me, and following Google's Material Design guidelines was another challenge in-and-of itself. Learning how activities, layouts, fragments, dialogs, and Android permissions along with storing user data were additional challenges.

Solutions: I am a really big fan of Google's Material Design, so I utilized their website (https://material.io) and learned about their styling rules. I also followed Google's Android Training Course to get an in-depth feel for Android application development. As for storing and retrieving info, I made sure it is as easy as possible for the user to export and import their data using Android's built-in internal storage.

Key Takeaways: I can say with confidence that learning Android Studio has been one of the greatest learning experiences as of writing this. It taught me so much more about Java, how users interact with software, storing information, writing legible code, and managing a product on a store page. I also gained experience working with others on a project, as I was lucky enough for my friend Charlie to create the store assets for Backup Code Manager. (Contact him at cmalachinski@outlook.com)
