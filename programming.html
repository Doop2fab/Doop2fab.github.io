Backtracking Rat-in-a-Maze Solver

This is a program which extends the Swing framework that solves Rat-in-a-Maze problems. It implements simple Backtracking by recursively determining if a space is safe to move to or not and, if the space in question is "un-safe", the rat backtracks. I have included 3 preset mazes, along with the ability to draw your own mazes.

Hunt The Wumpus

This is a program that reads in a file and generates a dungeon for the player to walk through. Your objective is to find the wumpus and shoot him, but not walk into the same room as him or run out of arrows. Also, avoid the traps! When you arrive in a room, you decide whether to move to an adjacent room or to shoot into an adjacent room. There are hints to what may lie beyond the room you are in, so watch out!
